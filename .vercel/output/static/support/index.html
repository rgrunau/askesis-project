Keep this work going

The Askesis Project is entirely reader-supported. There are no sponsors, no affiliate links, and no algorithmic considerations shaping what gets written. That freedom is only possible because of readers who find the work worthwhile enough to pay for it directly.

If these essays have been useful to you — or if you simply want to see more of them — consider making a contribution. Any amount is welcome, and nothing is behind a paywall.
